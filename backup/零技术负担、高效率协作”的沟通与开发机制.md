## 1. 角色定位清晰化：你指挥，我执行

你负责：

我负责：

拆解成技术任务（如“新增配置文件页面”、“用 token 代理 GitHub 请求”）

编写代码 / 提供文件 / 整理说明文档 / 打包脚本

告诉你每一步是干什么的，有什么影响，并保持所有改动都有可追踪记录
